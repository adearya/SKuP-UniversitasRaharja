Pendahuluan

Hi everyone!, saya ingin memperkenalkan diri saya, Nama saya Ade Arya Bimantara, biasa dipanggil Ade. Saya lahir di Tangerang atau lebih tepatnya di Kota Tangerang, saat ini pun masih tinggal di Kota Tangerang. Saat ini saya sebagai mahasiswa yang sedang menempuh pendidikan di Universitas Raharja yang mengambil jurusan Teknik Informatika dan berfokus pada Software Engineering, karena saya memiliki rencana karir menjadi Expert Programmer di masa depan.

Saat ini saya telah memasuki semester akhir dan juga sedang menjalankan Kuliah Kerja Praktek (KKP) di Alphabet Incubator (AI) dan menjadi salah satu anggota dari TimUR 17. Dari TimUR 17 saya tergabung ke dalam Grup Croissant. Baiklah mungkin itu saja sedikit perkenalan dari saya.

Saya menyukai Open Source dan Pemrograman bahkan kurang lebih sudah 5 tahun saya menekuni Pemrograman dan Open Source, alasan mengapa saya tertarik akan kedua hal tersebut karena menurut saya memiliki sebuah sistem yang kita buat sendiri itu sangat seru dan menarik. Sampai saat ini sudah banyak bahasa pemrograman yang telah saya pelajari seperti Python, Golang, Javascript, Java, Assembler, Visual Basic, C++, PHP, dan juga Shell Script yang sangat rumit. Selain itu saya juga mempelajari Sistem Operasi Linux dan Free Open Source Software (FOSS), tidak hanya itu bahasa Markdown dan bahasa Markup seperti HTML juga CSS. Saya juga mempelajari Version Control System (VCS), termasuk beberapa Framework dan juga DBMS dan masih banyak yang saya pelajari.
